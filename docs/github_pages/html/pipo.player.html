pipo.player.player module¶

Music Player.

class pipo.player.player.Player(bot)[source]¶

Bases: object

Manage music and Discord voice channel interactions.

Acts as facilitator to manage audio information while interacting with Discord. Maintains a music queue to which new audio sources are asynchronously added when calling play. A thread is used to stream audio to Discord until the music queue is exhausted. Whether such thread is allowed to continue consuming the queue is specified using can_play.
